Data

For gauging stations in Upper Austria we want to explain the low-flow index Q95 from different catchment characteristics. The variables are:

  • DBMSNR — database number (primary key, not used as regressor)
  • Q95 — low-flow index (l/s/km²) — response variable
  • H.MIN — altitude of the lowest point in the catchment (10⁻² m.a.s.l.)
  • H.MEAN — mean sea level within the catchment (10⁻² m.a.s.l.)
  • M.NEIG — mean terrain slope (%)
  • N.GES — total precipitation (10⁻² mm)
  • GEOL.BM / GEOL.QUA2 / GEOL.TER / GEOL.QUELL — fraction of geologic classes
  • BONU.ACK / BONU.GRU / BONU.WAL — fraction of land-use classes
  • SDENS — stream-network density

Stepwise Regression

Starting with the single most important variable from the full OLS model (lowest p-value), a bidirectional stepwise algorithm is applied.

# Identify the single most important predictor in the full OLS model
ols_pvals <- coef_summary[-1, "Pr(>|t|)"]   # drop intercept row
best_var   <- names(which.min(ols_pvals))
